AddressBook

Syntax

navigator.AddressBook.findContacts(filter, win, fail)

Example

navigator.AddressBook.findContacts({givenName: "Joe"}, win, fail);

function win(contacts)
{
   // Do something with contacts collection returned from the query
}

function fail()
{
  alert('nothing here');
}

Contact

The Contact object is what is returned by the findContacts call. This contains the following data. This object is a subset of the W3C Device API draft, and only the following is currently implemented at this time.

Code

var Contact = function() {
  this.givenNames = [];
  this.familyName = "";
  this.phones = [];
  this.email = [];
}
